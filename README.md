# 🚗 Car Sales/Registration Form

Um projeto focado na criação de um formulário de cadastro automotivo estruturado e esteticamente agradável. 

## 🚀 O que foi trabalhado:
* **Estrutura Semântica:** Uso correto de labels, fieldsets e tipos de input.
* **Estilização Customizada:** Foco em deixar os campos de texto e botões com um visual moderno (fugindo do padrão do navegador).
* **Responsividade:** Garantia de que o usuário consiga preencher o formulário facilmente tanto no PC quanto no telemóvel.

## 🛠️ Tecnologias
* HTML5
* CSS3 (Flexbox/Grid para alinhamento dos campos)
